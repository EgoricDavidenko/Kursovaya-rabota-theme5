Распакуйте архив javafx-sdk-25.0.1.rar в отдельную папку

Для запуска программы из jar архива необходимо в консоли написать следующую команду:
java --module-path "путь к папке, в которую вы распаовали архив\javafx-sdk-25.0.1\lib" --add-modules javafx.controls,javafx.graphics -jar путь к папке с jar-архивом\Rice_war.jar
